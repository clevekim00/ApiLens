# ApiLens

ApiLens는 REST, WebSocket, GraphQL을 하나의 워크플로우로 연결할 수 있는 데스크톱/웹 기반 API 도구입니다.

## 주요 기능 요약
- **Multi-Client**: REST / WebSocket / GraphQL 요청을 모두 지원합니다.
- **Workflow Editor**: 노드 기반 비주얼 에디터로 복잡한 API 시나리오를 자동화합니다.
- **Workgroup System**: 프로젝트 단위로 격리된 작업 공간을 제공합니다.
- **Advanced OpenAPI Import**: Swagger 문서를 필터링(Tags, Search)하여 필요한 API만 선택적으로 Import합니다.
- **Team Collaboration**: Workgroup을 파일(.json)로 Export/Import하여 팀원과 공유할 수 있습니다.
- **Cross Platform**: macOS, Windows 데스크톱 앱 및 Web 환경을 지원합니다.
- **Themes**: Light/Dark 테마를 지원합니다.

## Quick Start
1. **ApiLens 실행**: 데스크톱 앱을 설치하거나 웹 버전에 접속합니다.
2. **Workgroup 생성**: 사이드바 상단 `+` 버튼으로 새 작업 그룹을 만듭니다.
3. **OpenAPI Import**: Workgroup을 우클릭하고 `Import Swagger`를 선택하여 기존 API 명세를 불러옵니다.
4. **Request 실행**: 생성된 Request를 클릭하고 `Send` 버튼을 눌러 동작을 확인합니다.
5. **Workflow 생성**: `+ Workflow`를 눌러 여러 Request를 연결하고 자동화 로직을 구성합니다.
